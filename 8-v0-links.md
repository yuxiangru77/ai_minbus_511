## 📝 怎么用这些链接

### 给团队成员看

```
1. 把链接发到飞书群
2. 团队成员点击就能看到实时设计
3. 不需要任何技术背景
4. 可以截图、评论、提建议
```

### 修改设计

```
1. 点击链接进入 v0
2. 在右侧对话框输入修改要求
3. v0 实时生成新版本
4. 满意后点 "Copy code"
5. 把新代码替换到 2-/3-/4-/5-/6- 文件
```

### 重要：每次修改后

```
☐ 更新这个文件的"最后更新"日期
☐ 在 0-README.md 里写改了什么
☐ 通知技术团队
☐ 重新截图，更新 7-screenshots/
```
